# Adventure Works Auction System & Store Location Analysis

This repository contains a SQL-based project developed on top of the **Adventure Works** database, combining **database design, stored procedures, and business-oriented data analysis**.

The project was divided into two main components:

1. **Auction System Implementation**  
   Extension of the existing Adventure Works database to support an online auction system for stock clearance.

2. **Store Location Analysis**  
   A data-driven analysis to identify the two most promising cities in the United States for opening new brick-and-mortar stores without competing with existing high-value retail customers.

## Project Goals

The main objectives of this project were to:

- extend an existing relational database with a dedicated auction module
- design new tables and relationships to support auction operations
- implement stored procedures with validation, business rules, and error handling
- ensure data integrity and performance under higher transaction load
- analyse customer and sales data to support expansion decisions
- translate SQL outputs into actionable business recommendations

## Tech Stack

- **T-SQL**
- **Microsoft SQL Server**
- **Adventure Works**
- **Stored Procedures**
- **Relational Database Design**
- **Business Data Analysis**
